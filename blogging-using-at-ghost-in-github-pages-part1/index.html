Github Pages로 Ghost 이용하기 #1 - 설치

08 August 2016 on Ghost, Tutorial

요구사항

  • 윈도우 10 Pro

준비물

  • 윈도우용 Ubuntu Bash쉘 (이건 구글검색하는걸 추천한다)
  • Node LTS(4.x버전대)
  • Python 2(2.7버전대)
  • 끈기와 이해력

Node/Python 설치하기

긴이야기 안하겠다. 설마 개발하는사람들이 이것도 하나 모를까.

  • NodeJS LTS(4.x) https://nodejs.org/en/download/
  • Python 2.7 https://www.python.org/downloads/

Python 설치 후 추가설정하기.

Python을 추가로 설치하고 cmd를 켜고 python을 입력하면 죽어라 안된다.
그렇기에 명령어 한줄만 쳐주고 cmd창을 껐다켜자.

setx PATH "C:\Python27;C:\Python27\Scripts;C:\Python27\Lib\site-packages"

Buster 설치하기

Buster는 Ghost를 정적 페이지로 바꾸어 github pages에 올려주는데 도움을 주는 Python Package다. 그렇기에 pip를 통하여 설치가 가능하고,

Buster에 의존적인 lxml설치하기

Buster를 그냥 설치하려고하면 lxml부분에서 오류가 날것이다. C++컴파일러가 설치되어있지 않다고 나오거나, libxml2를 찾지못했다고 나온다. 그렇기에 python 패키지중에 wheel을 이용한다.

  1. 관리자권한으로 CMD를 연다.
  2. pip install wheel을 입력하여 wheel을 설치한다.
  3. 링크로 들어가서 lxml을 적당한 디렉터리에 받는다. 필자의 경우 lxml-3.6.1-cp27-cp27m-win32.whl을 이용했다.
  4. pip install (download-path)를 입력하여 설치한다.
    (download-path)는 당연히 .whl 파일을 받은 경로다.
  5. 설치완료 메세지가 뜨면서 설치가 완료된다.

lxml 설치가 완료되었다면 pip install buster를 입력하여 buster의 설치를 완료한다.

Buster를 위한 추가패키지 설치

  1. CMD/PowerShell을 열어 bash쉘로 진입한다.
  2. 다음과같은 명령어를 입력한다.
    apt-get install git wget

이렇게 되면 Windows 10에서 Ghost를 사용하기 위한 환경은 모두 갖추어졌다. 앞으로의 튜토리얼들은 Ghost와 Buster를 사용하는 방법/설정하는 방법에 대해서 다룰것이기 때문에 기존에 해당 툴들의 사용법을 숙지하고있는분들에 한해선 필요없는 내용들일것이다.

그렇지만 Buster → Bash / Ghost → Windows라는 개념만 꼭 숙지하도록 부탁한다. 아직 Windows Bash Shell이나 Buster를 이용한 Windows Path처리가 완벽하지 않기 때문에 이렇게 나누어서 작업해야된다.
